DigiTools Assinment

Digitools is a modern and smart digital tools platform where users can access multiple useful online tools in one place.The main goal of Digitools is to make everyday digital tasks easier, faster, and more time-saving. The project is built with a user-friendly UI, fast performance, and responsive design so that it works smoothly across mobile, tablet, and desktop devices.

Technology that I use: 

- React
- Tailwind css
- Daisy UI
- javascript (ES6)

3 Features about your project:
1.Multiple Useful Tools in One Platform – Users can access Starter Resume Pack, Professional CV Builder, Career Boost Pro, Freelancer Portfolio Kit, and more from a single website.
2.Responsive and User-Friendly Design – The platform is designed to work smoothly on mobile, tablet, and desktop devices with an easy-to-use interface.
3.Fast and Time-Saving Performance – Digitools helps users complete their digital tasks quickly and efficiently without needing multiple websites or applications.
